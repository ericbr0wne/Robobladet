Robobladet - A news web app built with React and Express.js

## RoboApp ## // Frontend

* cd roboApp
* npm install
* npm run dev

## RoboApi ## // Backend

* cd roboApi
* npm install
* _Required packages for login function_: npm install bcryptjs jsonwebtoken express-jwt
* node server.js
## RoboScraper ## // Webscraping news sites

* cd roboScraper
* dotnet build
* dotnet run
